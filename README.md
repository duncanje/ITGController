## ITGController

### Build
#### Prerequisites
- Install JDK
- Compile [D-ITG](http://traffic.comics.unina.it/software/ITG/download.php)
- Copy libITG.so to lib/

#### Linux
     ./build
     
#### Other
See javac commands in *build*

### Run
#### Linux
     ./ITGController <sender> <script>

#### Other
See java commands in *ITGController*
